
  pg123 Demonstrator File Set

  This set of files is for use with the demonstrator application. Program pg123
  reads the "pg123 Input Files" and outputs the "pg123 Output Files".
  
  pg123 Input Files

  pg123_demo.dat                 ! control data
  pg123_demo.d                   ! geometry
  pg123_demo.bnd                 ! boundary conditions
  pg123_demo.lds                 ! applied forces


  pg123 Output Files

  pg123_demo.res                 ! summary output
  pg123_demo.ensi.DISPL-000001   ! displacements
